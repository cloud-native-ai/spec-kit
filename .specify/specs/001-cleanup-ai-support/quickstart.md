# Quick Start Update (Draft)

## Installation

```bash
uv pip install -e .
```

## Initialization

Initialize a project with a supported AI assistant:

```bash
# GitHub Copilot (default recommended)
specify init my-project --ai copilot

# Qwen Code
specify init my-project --ai qwen

# opencode
specify init my-project --ai opencode
```

**Note**: Legacy providers (Claude, Gemini, etc.) have been removed to focus development on independent evolution.
