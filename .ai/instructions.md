## Project Overview
**Spec Kit (spec-kit)** 是一个面向 SDD 的 CLI 工具包：用模板与脚本生成 `.specify/` 结构与指令文件，驱动 `/speckit.*` 规范 → 计划 → 任务 → 实现的工作流，并支持 Copilot/Qwen/opencode。

> **Note**: This file is generated by `/speckit.instructions`. Custom content is preserved during updates.

## Core Concepts
- **Agents**：依赖 LLM 独立运行的智能体。Agent 可以结合上下文进行推理，并调用不同的 Skills 与 Tools，端到端完成多样化任务。
- **Skills**：用于特定任务的一组指令、脚本和资源，GitHub Copilot 可在相关场景下加载并执行。A skill is a directory containing at minimum a `SKILL.md` file.
- **Tools**：在 Agent 和 Skills 中可调用的、用于执行具体动作的工具集合，包含但不限于 MCP、Shell 函数、系统命令、项目脚本文件等。

三者关系：Agent 负责理解目标与编排执行；Skills 提供可复用的任务能力包；Tools 提供可被调用的底层执行能力。

## Tech Stack & Resources
- **Project Name**: spec-kit
- **Root Path**: /storage/project/cloud-native-ai/spec-kit
- **Language**: Python (项目声明 >=3.8；安装文档建议 3.11+)
